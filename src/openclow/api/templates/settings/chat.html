Switching Chat Platforms

OpenClow supports multiple chat platforms. When you switch from one to another:

  • Save the new provider configuration
  • User IDs will need to be updated to match the new platform
  • Restart the bot service: docker compose restart bot
  • Your projects and tasks are preserved — only the communication channel changes
